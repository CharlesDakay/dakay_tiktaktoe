# AWT Tic Tac Toe Game

This is a simple Tic Tac Toe game built using Java's AWT.

## How to run
- Compile: `javac TicTacToe.java`
- Run: `java TicTacToe`

## Features
- 3x3 Grid with Buttons
- Two-player turn system
- Winner detection
- Reset/New Game button
